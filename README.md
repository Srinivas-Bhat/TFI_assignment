# Teach For India.

#### Project where the user can signup and login for a particular user. User can also volunteer for TeachForIndia and register himself/herself. Admin can see the list of volunteers registered.

## Building Blocks.

1. HTML
2. CSS
3. React
4. MongoDB
5. Express
6. NodeJS
7. ChakraUI
8. React-Router-Dom
